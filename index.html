Getting Started

If you're new to our codebase, we'd recommend you start by checking out the following key components:

  • Processor
  • Gameplay::GameplayModule
  • The python Play class
  • SystemState
  • OurRobot

  • Logger
  • RobotFilter
  • VisionReceiver
  • NewRefereeModule
  • GameState
  • Planning::RRTPlanner
  • Planning::Path
  • MotionControl
